➢ A prototype module will be developed for the project. It includes individual PCB boards for all interfaces according to the block diagram. Every PCB will be inter-connected with jumper wires. ➢ In the camera using python scripts to detect the diver condition whether the eyes are open or closed. ➢ Here the eye detection will exist by using image processing and uses the Landmark Predictor algorithms. ➢ All the sensors are connected microcontroller. We have used Motor Driver L293 concept to control the DC Motor (Vehicle Engine). ➢ Whenever he feels sleepy automatically vehicle Engine will be OFF, meanwhile an alarm is generated. ➢ Whenever the driver feels sleepy then image will be captured and send as an email to the respective vehicle owners or cab owners so that he can able to take the actions. ➢ Alcohol detection sensor is used to detect drunk and drive. ➢ Here LCD is used to display the all the ongoing information
